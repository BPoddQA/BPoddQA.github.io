Only created cars can be checkin to the garage
By default, 8 cars are already created (show all created cars to display)
Select a car in the dropdown to check it in
Checked in cars will now appear in the check out dropdown
Select a car in the checkout dropdown to either check out a car, or view the cost of repairs
Output garage will output allt eh cars in the garage
Create car, will create a new car using the inputs
When using the admin console:
To create a car ==> create car (brand) (registration) (number_of_faults)
To output the list of cars in the garage ==> output garage
To check a car into the garage ==> check in (registration)
To check a car out of the garage ==> check out (registration)
To check the repair cost of a car in the garage ==> cost (registration)
If the brand contains a space, please use an underscore instead
When entering the registration number, do not include a space
